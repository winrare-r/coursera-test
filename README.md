# SETI Analyzer GUI

Графическое приложение на PyQt5 со stub-логикой анализа. Предназначено для выбора файла, запуска пайплайна и просмотра результатов в нескольких вкладках.

## Возможности
- Главный экран: выбор файла, история последних путей, выбор пресета анализа, статусная строка.
- Экран прогресса: индикатор, краткие статусы этапов, кнопка открытия лог-файла.
- Результаты: вкладки «Обзор файла», «Окна/Рейтинги», «Кандидаты» с таблицами и превью.
- Вкладка «Настройки»: параметры DBSCAN, опции предобработки, пути сохранения, тема интерфейса.
- Запуск stub-анализатора в отдельном потоке, обновление статусов и запись логов.
- Генерация демонстрационных графиков (водопад, карта активности, кластеры окон, спектр кандидатов) через matplotlib.

## Установка и запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m app.main
```

На Windows можно использовать `run_app.bat`, который запустит скомпилированный бинарь (если собран) или исходники.

## Сборка в единый файл (PyInstaller)
```bash
source .venv/bin/activate
python -m PyInstaller --clean --noconfirm app.spec
# или
./build_executable.sh
```
Готовый исполняемый файл появится в `dist/seti-analyzer/`.

## Структура
- `app/main.py` — точка входа, сборка UI, связывание сигналов.
- `app/analysis/processor.py` — заглушка анализатора, имитирует этапы и формирует примеры данных.
- `app.spec` — конфигурация PyInstaller с включением ресурсов и логов.
- `run_app.bat` — ярлык для запуска собранного приложения или исходников.
- `build_executable.sh` — удобный скрипт сборки.
