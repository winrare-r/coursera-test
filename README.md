# coursera-test

Минимальное описание приложения для обработки данных.

- Поток работы: выбор файла → пресет → запуск → просмотр вкладок с результатами.
- Сборка и запуск: Python 3.10+, установка зависимостей из `requirements.txt`, упаковка через PyInstaller.

Полное руководство и детали по настройкам см. в [docs/README.md](docs/README.md).
